flex

    container : 부모요소
    item : 자식요소
    컨테이너 요소에 아래와 같이 적용
    display :  flex

    justify-content : 중심축 기준 정렬
    align-items : 교차축 기준 정렬
    flex-wrap : 아이템 요소의 사이즈 유지, 부모 사이즈 벗어나면 다음줄로 넘김
    flex-direction : 중심축 설정
    align-content : 축 사이의 간격 조정

    아이템 적용 가능한 속성들
    align-self : 특정 아이템만 정렬 값 따로 지정 
    order : 순서 지정 (눈에 보이는것만 변경)
