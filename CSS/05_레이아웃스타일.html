포지셔닝

      position

      static : 기본값
                HTML 작성 순서대로 쌓임

      relative : 기존 위치를 기준으로 이동
                (다른 무언가가 침범 불가)

      absolute : 부모 위치를 기준으로 위치 선정
                (부모의 조건 : position 값이 static이 아니어야 함)
                (부모조건에 맞지 않으면, 상위요소를 찾아감...반복)
                (찾다가 없으면 html 요소까지 올라감)
                유령이 됨 (==다른 요소와 충돌 없음, 기존 자리 차지 안함)

      fixed : 화면 기준 위치선정
              유령이 됨 (==다른 요소와 충돌 없음, 기존 자리 차지 안함)

      sticky : 조건부 fixed
                기본적으로는 자기 자리 지킴
                근데, 화면 기준으로 자리잡을 수 있으면 fixed가 됨
                부모요소를 벗어날 수 없음
